HAAM Summer School – Introduction to R

Files:

1. HAAM_intro_to_R.qmd
   Student-facing Quarto worksheet. Contains explanations, runnable code chunks, and empty exercise chunks.

2. HAAM_intro_to_R.R
   Clean script version. Contains the full code, including example answers.

3. HAAM_intro_to_R_answers.qmd
   Answer key version. Share after the workshop or keep for teaching.

Dataset:

This workshop uses the built-in iris dataset, so no external files are required.

Required package:

- tidyverse (preinstalled on the VMs)
